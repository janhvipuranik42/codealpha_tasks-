# CodeAlpha Python Internship Tasks

This repository contains Python programming tasks completed as part of the **CodeAlpha Python Internship Program**.  
All projects are beginner-friendly and demonstrate core Python concepts.

---

## Tasks Completed

### Task 1: Hangman Game
- Text-based Hangman game where the player guesses a word letter by letter
- Uses a predefined list of words
- Limits incorrect guesses
- Implements loops, conditional statements, and string operations

---

### Task 2: Stock Portfolio Tracker
- Calculates total investment value based on user input
- Uses a hardcoded dictionary for stock prices
- Demonstrates dictionaries, loops, and basic arithmetic operations

---

### Task 4: Basic Chatbot
- Simple rule-based chatbot
- Responds to predefined user inputs like greetings and exit commands
- Uses if-elif conditions and loops for continuous interaction

---

## Technologies Used
- Python
- Core programming concepts (loops, conditions, functions, data structures)
